Canvas Text Box Examples

Example 1 - Simple

Simple 350px wide text box in a 400px wide canvas.


let text = 'Lorem ipsum dolor sit amet, ...';

let canvas1 = document.getElementById("example1-canvas");
let options1 = {
    textFillStyle: '#000000',
    fontSize: 22,
    lineHeight: 26,
    width: 350,
    startX: 26,
    startY: 26
};
canvasTextBox.draw(canvas1, text, options1);

Example 2 - Bottom aligned

Same as in example 1 - only bottom aligned.

In this example, the text box is aligned to the bottom of the canvas, and startX and startY properties actually refer to the bottom left point of the text box:

  • startX being the distance from the left side of the canvas
  • startY being the distance from the bottom side of the canvas

let text = 'Lorem ipsum dolor sit amet, ...';

let canvas2 = document.getElementById("example2-canvas");
let options2 = {
    textFillStyle: '#000000',
    fontSize: 22,
    lineHeight: 26,
    width: 350,
    startX: 26,
    startY: 26,
    bottomAligned: true
};
canvasTextBox.draw(canvas2, text, options2);

Example 3 - Background under text

Same as in example 2 - only added background.

In addition to options introduced in the previous example, useBackground option is added and set to true. It uses default background color, if not defined otherwise using backgroundFillStyle option.

Note: In this case background is visible only under the text itself, it does not take the entire area of the text box. 


let text = 'Lorem ipsum dolor sit amet, ...';

let canvas3 = document.getElementById("example3-canvas");
let options3 = {
    textFillStyle: '#000000',
    fontSize: 22,
    lineHeight: 26,
    width: 350,
    startX: 26,
    startY: 26,
    bottomAligned: true,
    useBackground: true
};

Example 4 - Background under entire text box

Same as in example 3 - only background cover entire text box.

In addition to options introduced in the previous example, fullWidthBackground option is added and set to true.

Example 5 - centered text on full width background

This example demonstrates the use of, textAlign option with value set to center. Default value of this option is left 


let text = 'Lorem ipsum dolor sit amet, ...';

let canvas5 = document.getElementById("example5-canvas");
let options5 = {
    textFillStyle: '#000000',
    fontSize: 22,
    lineHeight: 26,
    width: 350,
    startX: 26,
    startY: 26,
    bottomAligned: true,
    useBackground: true,
    fullWidthBackground: true,
    textAlign: 'center'
};
canvasTextBox.draw(canvas5, text, options5);

Example 7 - meme generator

Font style and positioning in a use case that resembles a meme generator. 


let canvas7 = document.getElementById("example7-canvas");
let options7 = {
    fontFamily: 'Impact, Charcoal, sans-serif',
    textFillStyle: '#FFFF00',
    fontSize: 40,
    fontWeight: 'bold',
    lineHeight: 52,
    width: 400,
    startX: 0,
    startY: 52,
    bottomAligned: false,
    useBackground: false,
    textAlign: 'center'
};
let options71 = Object.assign({}, options7);
options71.bottomAligned = true;
options71.marginBottom = 20;
canvasTextBox.draw(canvas7, 'Y U NOT USE THIS', options7);
canvasTextBox.draw(canvas7, 'FOR MEME GENERATOR', options71);
